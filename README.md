# Portfolio Website (Dark Tech Theme)

A simple personal portfolio website built using **HTML, CSS, and JavaScript**.

## Pages
- `index.html` — Home
- `about.html` — About me / interests / skills
- `portfolio.html` — Projects with GitHub links (cards)
- `resume.html` — Resume (two-column layout) + view/download
- `contact.html` — Contact page (email/phone/LinkedIn)

## How to run
Just open `index.html` in your browser (or use VS Code Live Server).

## Tech
HTML • CSS • JavaScript • Font Awesome (icons)
